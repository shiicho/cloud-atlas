# Docker 容器技术：从入门到实践

> 从零开始系统学习 Docker 容器技术，掌握现代应用部署的核心工具。

## 课程特色

- **理念先行**：理解容器为什么这样设计
- **本地优先**：无需云资源，本机即可学习
- **反模式教学**：展示常见错误，再教正确做法
- **精简聚焦**：只教必要的，高级话题留给进阶课程

## 课程大纲

| 课程 | 主题 | 时长 |
|------|------|------|
| [00](./00-concepts/) | 概念导入：Docker 架构与设计 | 20 min |
| [01](./01-first-container/) | 安装配置与第一个容器 | 35 min |

## 前置要求

- **推荐**：基本的 Linux 命令行操作经验
- **推荐**：Git 基础知识（用于管理 Dockerfile 和 Compose 文件）
- **可选**：Bash 脚本基础（用于容器入口脚本）

## 学习环境

Docker 可以完全在本地学习，无需云资源：

- **Docker Desktop**（推荐）：Windows / macOS 最简单的方式
- **Docker Engine**：Linux 原生运行
- **云端环境**：使用已有的 EC2 实例
