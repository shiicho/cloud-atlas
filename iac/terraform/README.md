# Terraform 实战课程

> **从零开始，用 Terraform 管理 AWS 基础设施**

---

## 课程特色

- **味道优先**：先运行代码，再理解原理
- **真实场景**：基于 AWS 生产环境实践
- **可复现**：完整示例代码，克隆即用
- **双语支持**：中文讲解 + 英文术语

## 前置要求

- AWS 账户（有基本的 IAM 权限）
- AWS CLI 已配置
- 基本的命令行操作能力
- 了解 AWS 核心服务（EC2, S3, VPC）

---

## 实验环境

本课程提供 CloudFormation 模板，一键部署预装 Terraform 的开发环境：

- EC2 实例（t3.small）
- 预装 Terraform、AWS CLI、Git
- SSM Session Manager 访问（无需 SSH Key）
- VS Code Remote 兼容

详见 [环境准备](./00-concepts/lab-setup.md)。

---

## 示例代码

每课都有配套代码，位于 `code/` 目录：

```bash
git clone --filter=blob:none --sparse https://github.com/shiicho/cloud-atlas ~/cloud-atlas
cd ~/cloud-atlas && git sparse-checkout set iac/terraform
cd ~/cloud-atlas/iac/terraform/01-first-resource/code
terraform init
terraform plan
```
