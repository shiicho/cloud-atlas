# NAT 与穿透（NAT & Traversal）

**TL;DR**：NAT 让多台设备共享一个公网 IP。副作用是外部无法主动连接内部设备。「穿透」就是绕过这个限制的方法。

## 什么是 NAT？

家里/公司多台电脑，但只有一个公网 IP。路由器用 NAT 把内网地址（192.168.x.x）转换成公网地址。

```
内网设备 (192.168.1.10) ──► NAT 路由器 ──► 互联网 (显示为 203.0.113.1)
```

## NAT 的限制

- ✅ 内部可以访问外部（你能上网）
- ❌ 外部无法主动连接内部（别人连不进来）

## 常见穿透方法

| 方法 | 原理 |
|------|------|
| **端口转发** | 路由器设置：公网端口 → 内网设备 |
| **反向连接** | 内部设备主动连外部，保持通道 |
| **VPN** | 建立隧道绕过 NAT |

## 记住这个

> 如果你的服务在 NAT 后面需要被外部访问，要么配置端口转发，要么让它主动连出去。
